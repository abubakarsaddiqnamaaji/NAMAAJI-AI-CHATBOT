#  Namaaji AI Chatbot

Namaaji AI is an educational AI-powered chatbot, built to help students learn, ask questions, and understand topics in a simple and interactive way.

It supports both **English and Hausa** and is designed with a modern UI for better user experience.

<img width="1254" height="1254" alt="logo" src="https://github.com/user-attachments/assets/6991ceb4-7fb7-429a-9841-fc00f535d13c" />

##  Features

-  AI-powered chat system
-  English + Hausa and another languages support
-  Fully responsive design
-  Fast and modern UI
-  Markdown response rendering
-  Real-time AI responses via API
-  Educational assistant focus

---

## 🛠️ Built With

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- OpenRouter AI API
- Marked.js (Markdown rendering)

---

## ⚙️ How It Works

1. User types a message
2. Message is sent to AI API (OpenRouter)
3. AI processes the request
4. Response is returned and displayed in chat UI
5. Markdown is rendered into clean formatted text

---

## Important Note

This project uses an API key for AI requests.  
For production use, it is recommended to move the API call to a backend server to protect the key.
